🎯 CORE FILES (Attach These to Claude Code)

When spawning a Claude Code session, attach:
  1. CLAUDE-CODE-CONTRACT.md               (standards — always)
  2. CODING-CONTEXT.md                      (language rules — always)
  3. TASK-BRIEF.md                          (your task — copy from template)
  4. The matching prompt for your task type:
     • Generate:        prompts/PROMPT-CLAUDE-CODE-MASTER.md
     • Add Function:    prompts/PROMPT-ADD-FUNCTION.md
     • Refactor:        prompts/PROMPT-REFACTOR.md
     • Debug:           prompts/PROMPT-DEBUG.md
     • Partial Rewrite: prompts/PROMPT-PARTIAL-REWRITE.md
  5. THINKING-FRAMEWORKS.md                (optional — for complex/novel tasks)

🚀 HOW IT WORKS

1. Pick task type
   (generate / add function / refactor / debug / partial rewrite)

2. Fill TASK-BRIEF.md
   (task type, requirements, mode-specific sections)

3. Attach 4 files to Claude Code
   (CONTRACT, CONTEXT, BRIEF, matching PROMPT)

4. Claude reads contract + mode prompt
   (knows exact quality expectations and constraints)

5. Claude produces output
   (meeting all mode-specific standards)

6. You verify
   (make test, make lint, git status)

7. You ship
   (code is production-ready)
