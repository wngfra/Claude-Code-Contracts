QUALITY STANDARDS (Always Enforced)

Every Claude Code project must deliver:
  ✓ 100% type hints on public APIs
  ✓ ≥80% test coverage (all tests pass)
  ✓ Benchmarks for hot paths
  ✓ Zero TODOs (complete code)
  ✓ Zero linting warnings
  ✓ Zero type errors
  ✓ Security audit clean
  ✓ CI configuration included
  ✓ Working setup script (./setup.sh)
  ✓ Accurate README with examples
  ✓ Ready to git commit immediately

HOW IT WORKS

1. Fill TASK-BRIEF.md
   (your project name, language, requirements, performance budget)

2. Attach 4 files to Claude Code
   (CONTRACT, CONTEXT, BRIEF, MASTER-PROMPT)

3. Claude Code reads contract
   (knows exact quality expectations for your language)

4. Claude Code generates code
   (meeting all standards automatically)

5. You verify
   (make test, make lint, make bench, git status)

6. You ship
   (code is production-ready, no iteration)
